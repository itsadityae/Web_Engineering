HTML Text Formatting Tags

This page lists and describes all text formatting tags in HTML, along with examples of their usage.

Tag Description Example
<b> Defines bold text. This text is bold
<strong> Defines important text (usually bold). This text is important
<i> Defines italicized text. This text is italicized
<em> Defines emphasized text (usually italic). This text is emphasized
<u> Defines underlined text. This text is underlined
<s> Defines strikethrough text. This text is strikethrough
<mark> Defines highlighted text. This text is highlighted
<small> Defines smaller text. This text is smaller
<sub> Defines subscript text. H2O
<sup> Defines superscript text. X2
<code> Defines code text (monospace font). This is code text
<pre> Defines preformatted text. 
This is preformatted text
with line breaks.
<q> Defines a short inline quotation. This is a short quote
<blockquote> Defines a long block quote.
This is a longer quote that stands alone as a block.
<abbr> Defines an abbreviation or acronym. WHO
<cite> Defines the title of a work (e.g., book, movie). The Great Gatsby
<del> Defines deleted text. This text is deleted
<ins> Defines inserted text. This text is inserted
